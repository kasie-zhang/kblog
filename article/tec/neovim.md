## 1.what is Neovim

[Neovim](https://neovim.io/) 自己的定位是 Hyperextensible Vim-based text editor。

Neovim 是 Vim 传统中的重构，不是重写，更像是一种扬弃，摒弃了 Vim 中存在的历史老旧问题，将 Vim 中优秀的部分剥离出来并组成 Neovim。

就目前来看，Neovim 在 GitHub 上拥有更高的热度，有更活跃的开发者环境。
